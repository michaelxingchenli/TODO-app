## Available Scripts

### `npm run start-dev`<br />
Open [http://localhost:8080](http://localhost:8080) to view it in the browser.

Runs the app in development mode

### `npm run server`

Runs server [port 8888](http://localhost:8888)

### `npm run dev`

Runs the frontend only
